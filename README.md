# telemetric server
## possible uploads
- `ragequit`
    - `level` als int (1 oder 2)
    - `time` als float (timestamp in milisekunden)
- `win`
    - `level` als int (1 oder 2)
    - `time` als float (timestamp in milisekunden)
- `death`
    - `level` als int (1 oder 2)
    - `time` als float (timestamp in milisekunden)
    - `positionx` als float
    - `positiony` als float

- `get_heatmaps`
    - zeigt heatmaps an
- `update_heatmaps`
    -- updated heatmaps mit neuen Werten
